# House_Price_Prediction

This project predicts house prices using regression models.

## Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Evaluation Metrics
- RMSE
- MAE

## Features
- Exploratory Data Analysis (EDA)
- Missing value handling
- Log transformation
- One-hot encoding
- Feature scaling
- Residual analysis
- Saved model (.pkl)

## Files
- house_price_prediction.ipynb
- house_price_model.pkl
- scaler.pkl

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open house_price_prediction.ipynb in Jupyter Notebook.
3. Run all cells.

## Example Prediction
The final notebook cell demonstrates how to load the model and make predictions.
